Configuration

To bring up the "Settings" dialog, click on the spanner button.

Hopefully this dialog is fairly self-explanatory but just to be explicit, the settings are described below.

ItemDescription
Composition This is the absolute path to the composition file. It may be set using the file browser button to the right
Server This is the absolute path to the server executable. It may be set using the file browser button to the right
Minimum Note Length This sets the fastest speed of ringing - i.e. the value of the rightmost position on the speed slider
Maximum Note Length This sets the slowest speed of ringing - i.e. the value of the leftmost position on the speed slider
Initial Note Length This sets the initial position of the speed slider. A low value sets a high speed.
Midi Channel This sets the midi channel used by the server
Server Port This sets the tcp port that both the client and server share. The value is not critical but
make sure it doesn't conflict with other applications you may be running
